# 🚀 PHP Smart Files v1.0.0 Release Notes  
*Released on 2025-06-03*

## ✨ New Features

- 🎉 Initial release of PHP Smart Files extension with powerful capabilities:
  - Create PHP files effortlessly with automatic namespace detection
  - Multi-language UI support (English, German, French, Spanish, Russian)
  - Integrated into VS Code Explorer context menu for smooth file management
  - Intelligent handling of PHP namespaces including partial-qualified references
  - Advanced refactoring for files, directories, and namespaces
  - Improved performance and stability during large refactorings
  - Enhanced user experience with localized messages and progress indicators

## 🔧 Improvements

- Updated all dependencies to latest versions for better compatibility and security
- Completed extensive testing to ensure stability and reliability for first release
